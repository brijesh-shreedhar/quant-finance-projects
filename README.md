# Quantitative Finance & Systematic Investment Research

This repository contains Python-based quantitative research projects focused on analyzing equity markets, evaluating return drivers, and exploring portfolio construction techniques through a systematic, data-driven approach.

The work emphasizes financial intuition combined with statistical analysis, time-series modeling, and portfolio analytics to better understand how investment strategies behave across different market conditions.

---

## Research Focus

- Equity return analysis and time-series behavior  
- Factor exposure and return decomposition  
- Portfolio construction and optimization  
- Statistical modeling and regression analysis  
- Risk evaluation including volatility, drawdowns, and return stability  

---

## Project Overview

### 1. Healthcare Equity Time-Series Analysis
- Analyzed historical price behavior of healthcare equities using time-series techniques  
- Evaluated return distributions, volatility patterns, and trend characteristics  
- Assessed consistency of returns and sector-specific risk dynamics  

### 2. PCA Factor Analysis & Portfolio Insights
- Applied Principal Component Analysis (PCA) to identify underlying drivers of equity returns  
- Measured explained variance and interpreted dominant factors across assets  
- Evaluated how factor structure impacts diversification and portfolio behavior  

### 3. Portfolio Optimization via Monte Carlo Simulation
- Simulated a wide range of portfolio allocations to evaluate risk-return tradeoffs  
- Identified optimal portfolios based on Sharpe ratio and volatility  
- Compared allocation outcomes under different assumptions and constraints  

### 4. S&P 500 Returns Regression Analysis
- Built regression models to examine relationships between market variables and equity returns  
- Tested statistical significance and stability of predictors  
- Explored how different factors influence market performance over time

## Analytical Approach

Each project follows a structured research workflow:

1. Data collection and preprocessing  
2. Exploratory data analysis  
3. Model development or statistical testing  
4. Interpretation of results in a financial context  

The focus is on understanding **what drives returns and risk**, rather than purely building predictive models.
