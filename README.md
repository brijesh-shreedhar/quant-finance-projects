# Quant Finance & Investment Strategy Projects

This repository showcases Python-based projects focused on financial analysis, portfolio evaluation, and machine learning applications in investment strategy.

## Projects

### Portfolio Optimization (Monte Carlo)
- Simulated thousands of portfolios to analyze risk-return tradeoffs
- Identified optimal allocations and efficient frontier dynamics

### PCA Factor Analysis
- Applied principal component analysis (PCA) to financial datasets
- Identified key factors driving asset behavior

### S&P 500 Returns & Regression Analysis
- Analyzed equity returns and statistical relationships across securities
- Applied regression techniques to identify key drivers

### Healthcare Equity Time Series Analysis
- Evaluated sector-specific stock trends over time
- Built time-series workflows using Python

## Tools & Technologies
- Python (pandas, NumPy, scikit-learn)
- Data analysis & visualization
- Machine learning (regression, classification)

## Purpose
These projects demonstrate practical applications of quantitative analysis and modeling relevant to investment strategy and finance roles.
